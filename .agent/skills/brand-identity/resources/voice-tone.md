# Redacción: Guía de Voz y Tono

Al generar texto, adhiérete a la personalidad de Aceros Arequipa.

## Personalidad de la Marca

| Atributo | Descripción |
|----------|-------------|
| **Sólida y Segura** | Transmitimos confianza absoluta. Somos líderes en el mercado. |
| **Experta y Educativa** | Enseñamos cómo construir bien. Somos el referente técnico. |
| **Cercana y Peruana** | Hablamos de tú a tú con el "Maestro de obra", el ingeniero y el autoconstructor. |
| **Directa** | Sin rodeos, lenguaje claro y útil. |

## Reglas de Gramática y Mecánica

* **Encabezados:** Usa mayúsculas tipo título (Title Case) para mayor impacto visual.
* **Términos Clave:** Capitaliza palabras como "Seguridad", "Calidad" y "Construcción" cuando se usen como conceptos de marca.
* **Signos de exclamación:** Úsalos con moderación, solo para frases de aliento o eslóganes (ej. "¡Seguridad ante todo!").

## Guía de Terminología

| ❌ No Usar (Muy formal/Genérico) | ✅ Usar (Estilo Aceros Arequipa) |
| :--- | :--- |
| "Cliente" | "Socio" o "Amigo constructor" |
| "Varilla de metal" | "Fierro" o "Acero" |
| "Lugar de trabajo" | "La obra" |
| "Utilizar" | "Usar" |
| "Resistencia sísmica" | "Seguridad antisísmica" |
| "Ayuda" | "Capacitación" o "Asesoría" |
| "Comprar" | "Construir con..." |
| "Producto" | "Solución" o "Material" |

## Ejemplos de Tono

### Mensajes de Bienvenida
* ❌ *Incorrecto:* "Bienvenido a nuestra plataforma de servicios."
* ✅ *Correcto:* "¡Bienvenido, Socio! Empecemos a construir juntos."

### Formularios
* ❌ *Incorrecto:* "Por favor, introduzca sus datos para procesar la solicitud."
* ✅ *Correcto:* "Ingresa tus datos para empezar a construir juntos."

### Mensajes de Error
* ❌ *Incorrecto:* "Error: La operación no pudo completarse."
* ✅ *Correcto:* "Algo salió mal. Revisa los datos e intenta de nuevo."

### Llamadas a la Acción
* ❌ *Incorrecto:* "Haga clic aquí para más información."
* ✅ *Correcto:* "Conoce más sobre Seguridad antisísmica."

### Contenido Educativo
* ✅ *Correcto:* "La seguridad de tu obra es lo primero. Revisa los planos aquí."
* ✅ *Correcto:* "Aprende a calcular el fierro que necesitas con nuestra Asesoría técnica."

## Frases Clave de la Marca

Usa estas frases cuando sea apropiado:
* "La seguridad de un fierrazo"
* "Construyendo el Perú"
* "Calidad que resiste"
* "Seguridad antisísmica garantizada"
